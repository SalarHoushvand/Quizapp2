Derived indenties and demorgan's laws

There are a number of binary operations defined on sets. There are properties of these operations that imply the truth of statements. These implied true statements are called identities.

The following outlines some of the known identites first by operation and then the statements that involve more than one operation. Some of the identites were already introduced in the sections for the operation. In each of the following, let \(A, B\) and \(C\) be sets. The Universal set is \(U,\) while the Empty set is \(\emptyset.\) Remember, a statement is not true if there exists one example for which the statement is not true. You cannot prove the truth of a statement by showing the statement holds for one example.

Set Union:

identity law, \(A \cup U = U\)

domination law, \(A \cup \emptyset = A\)

idempotent law, \(A \cup A = A\)

complement law, \(A \cup A^{c} = U\)

commutative law, \(A \cup B = B\cup A\)

associative law, \($ ( A \cup B )\cup C = A\cup ( B \cup C )\)

Set Intersection:

identity law, \(A \cap U = A\)

domination law \(A \cap \emptyset = \emptyset\)

idempotent law \(A \cap A = A\)

complement law \(A \cap A^{c} = \emptyset\)

commutative law \(A \cap B = B \cap A\)

associative law \($ ( A \cap B )\cap C = A \cap ( B \cap C )\)

Set Symmetric Difference:

\[A \Delta U = \emptyset\] \[A \Delta \emptyset = A\] \[A \Delta A = \emptyset\] \[A \Delta A^{c} = A\] \[A \Delta B \neq B \Delta A\] \[( A \Delta B )\Delta C = A \Delta ( B \Delta C )\]

Cartesian Product:

\[A \times \emptyset = \emptyset A= \emptyset\] \[A \times ( B \cup C ) = ( A \times B ) \cup ( A \times C )\] \[A \times ( B \cap C ) = ( A \times B ) \cap ( A \times C )\]

\(A \times ( B \times C ) = ( A \times B ) \times C\)$

Absorption Laws regarding Set Union and Intersection:

\[A \cap (A \cup B) = A\] \[A \cup (A \cap B) = A\]

Distributive Laws regarding Set Union and Intersection:

\[A \cap (B \cup C) = (A \cap B) \cup (A \cap C)\] \[A \cup (B \cap C) = (A \cup B) \cap (A \cup C)\]

Intersection distributes over Symmetric Set Difference:

\[A\cap \left ( B\Delta C \right )=\left ( A\cap B \right )\Delta \left ( A\cap C \right )\]

Intersection written in terms of Set Difference

\[A \cap B = A - (A - B)\]

The Complement distributes over Set Union, Set Intersection and Set Difference and Set Symmetric Difference: \(U^{c} = \emptyset\)

\[(\emptyset)^{c} = U\] \[(A \cap B)^{c} = A^{c} \cap B^{c}\] \[(A \cup B)^{c} = A^{c} \cup B^{c}\] \[(A^{c})^{c} = A\] \[A \Delta B = A^{c} \Delta B^{^{c}}\] \[\emptyset^{^{c}} = U\] \[U^{c}=\emptyset\]

DeMorgan’s Laws:

\[(A \cap B)^{c} = A^{c} \cap B^{c}\] \[(A \cup B)^{c} = A^{c} \cup B^{c}\]

Distributive laws: \(A\cap \left ( B\cup C \right )=\left ( A\cap B \right )\cup \left ( A\cap C \right )\)

It is a very good idea to experiment with a statement’s validity with concrete examples before trying to prove the statement. You just might get lucky and find either a hint to the sequence of statements that would provide a proof or a counter-example to the statement. Let \(A, B\) and \(C\) be arbitrary finite sets.

We can generalise De Morgan’s laws to any number of sets \(\left ( \bigcup_{n} S_{n}\right )^{_{c}}=\bigcap_{n}S_{n}^{c}\) \(\left ( \bigcap_{n} S_{n}\right )^{_{c}}=\bigcup_{n}S_{n}^{c}\) where \(S_n, n=1,2,3,...,n\) are arbitrary sets.

A proof of DeMorgan’s laws \(( A \cap B )^{c} = A^{c}\cup B^{^{c}}\).

To prove that \(( A \cap B )^{c} \subseteq A^{c}\cup B^{^{c}},\) let \(x \in ( A \cap B )^{c}.\) Then \(x \notin ( A \cap B ).\) This implies that either \($ x \notin A \text{ or } x\notin B.\) We then have that \(x \in A^{c}\cup B^{c}.\) Now let \(x \in A^{c}\cup B^{^{c}}.\) Then \(x \in A^{c } \text{ or } x \in B^{c}\) which means that \(x \in A^{c}\cup B^{c} \blacksquare\)
