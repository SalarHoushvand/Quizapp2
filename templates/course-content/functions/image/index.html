Functions (image)

The Image set for a rule \(r\) with Domain set \(D\) and Range set \(R\) is the collection of all elements \(y\) in \(R\) such that there exists an \(x\) in \(D\) with \(r(x) = y.\) The element \(y\) in this definition is referred to as the image of \(x\) under \(r.\) Using our set notation, the Image set for the tuple \((r, D, R),\) is the subset of \(R\)

\[Im(R) = \{y \vert y \in R \text{ and } y = r(x) \text{ for some } x \in D\} \subseteq R\]

Example 1 For the rule \(r_1\) with Domain set \(D =\{ -2, -1, 0, 1, 2, \}\) and Range set \(R = \{ -1, 1\},\) define \(r_{1}(x)\) with the statement â€œif \(x\) is in \(Z\) then \(r_{1} (x) = 1\) for \(x > 0\) and \(r_{1}(x) = -1\) for \(x \leq 0.\) The Image set for \(r_1,\) \(Im(r_1),\) is the collection of images of the elements in \(D\) under \(r_1.\)

\[Im(r_1) = \{ r_1(-2), r_1(-1), r_1(0), r_1(1), r_1(2) } = \{ -1, 1\} = R.\]

Example 2 Define the rule \(r_{2}\) from the discrete set of all positive integers \(Z^+ = \{ 1, 2, 3, \ldots \}\) to the discrete set of all rational numbers \(\mathbb{Q} = \{q \vert q = \frac{x}{y} \text{ with } x,y \in \mathbb{Z} \text{ and } y \neq 0\}\),\(with the following:\) r_{2}(x) = \frac{1}{x}.\(The Image set for\)r_2\(is a subset of\)\mathbb{Q}\(with\)Im(r_2) = {q \vert q = \frac{1}{x}, x \in \mathbb{Z^+}}.$$

Why do we need both a Range set and an Image set for a rule? Rules cannot always be explicitly defined. There are fields of study where the rule involves randomness. For example, an experiment is set up with a jar containing 6 pieces of candy and one child is given permission to take as much of the candy as they would like. The rule maps to each child the number of pieces of candy the child takes, and only after the experiements are conducted do we know the image for each child under the rule. The Domain set for this situation is the set of all children in repeated experiments. The Range set is the possible number of pieces of candy the child takes from the jar, \(\{0, 1, 2, \ldots, 6\}.\) Say the experiment is repeated for four children with the outcomes (number of pieces of candy) recorded 1 piece, 0 pieces, 5 pieces, 1 piece. The Image set (in pieces of cancy) is \(\{1, 0, 5\}.\)
