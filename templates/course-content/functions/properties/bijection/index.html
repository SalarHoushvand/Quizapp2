Function properties (bijection)

A function \((D_f, R_f, f)\) is said to be a bijection if it is both injective and surjective, meaning that every \(y\) in the Range set of \(f,\) \(R_f,\) is the image of exactly one element \(x\) in Domain set of \(f,\) \(D_f\) so that is a one-to-once correspondence between the elements of \(D_f\) and the elements of \(R_f.\)

Example 4 Take the Domain set to be \(D_f = Z = { \ldots, -2, -1, 0, 1, 2, \ldots }\) and the Range set \(R_f = N = { 0, 1, 2, \ldots }\). Define \(f\) with the statement \(f(x) = -2x-1\) if \(x < 0\) and \(f(x) = 2x\) if \(x \geq 0.\) To show that \(f\) is onto, or rather surjective, we note that if \(y\) is in \(R_f,\) then \(y\) is either even or odd. In the case that \(y\) is even, there is an integer \(z\) so that \(y = 2z.\) It follows that since \(y \geq 0,\) \(z \geq 0.\) By the rule, \(f(z) = 2z = y.\) In the case that \(y\) is odd, then there exists an integer \(z\) with \(y = 2z -1.\) Since \(y \geq 1,\) \(2z-1 \geq 1\) and \(z \geq 1.\) Using the properties of multiplication, addition and the ordering of the integers, \(-z \leq -1..\) Since \(-(-z) = z,\) we have an integer \(\hat{z} = -z \leq 1 < 0,\) with \(f(\hat{z}) = y = 2(-\hat{z}))-1 = -2\hat{z} - 1.\) To prove that \(f\) is one-to-one, or injective, assume there is an \(y\) in \(R_f\) and \(x_1, x_2\) in \(D_f\) with \(x_1 \neq x_2.\) If both \(x_1\) and \(x_2\) are greater than \(0,\) then \(x_1 \neq x_2\) implies \(2x_1 \neq 2x_2.\) So that \(f(x_1) \neq f(x_2).\) If both \(x_1\) and \(x_2\) are less than \(x_1 \neq x_2\) implies that \(2x_1 - 1 \neq 2x_2 -1.\) And again, \(f(x_1) \neq f(x_2).\) The only other case is if \(x_1\) and \(x_2\) are both \(0,\) but we are only out to test whether \(f\) is injective, meaining that if two elements in the Domain set are not equal, then their images are not equal. The function \(f\) is thus a bijection.
