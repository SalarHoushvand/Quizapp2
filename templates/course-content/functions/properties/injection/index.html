Function properties (injection)

Let \((r, D, R)\) be a tuple representing a rule \(r\) with Domain set \(D\) and Range set \(R.\) Then, if to each element \(x\) in \(D\) their is a unique image in the Range set \(R,\) the rule is considered well-defined. This does not restrict the case where an element in the Range set has more than one “pre-image” in the Domain set. For example, the tuple \((r_1, \{-2, -1, 0, 1, 2\}, \{-1, 1\})\) where \(r_1(x) = 1\) for all \(x\) is a well-defined function since for every \(x\) in the Domain set there is only one image in the Range set. However, the set of all pre-images of the element \(1\) in the Range set is all of the Domain set. In other words, \(1\) has 5 pre-images. In this module, we identify rules that behave following a “1-1” principle.

A rule is an injection if the size of the pre-image set for every element in the Range is at most 1. An equivalent definition for an injection follows. A rule \((r, D, R)\) is an injenction if for every element \(y\) in the Range set \(R\), there is at most one element in the Domain set \(D\) that maps onto \(y\) under the rule.

From the definition of an injection, for those elements in the Range set \(R\) that have a pre-image, this pre-image is unique. The element \(x\) in the Domain set with \(r(x)= y\) is called the inverse of y under \(r\) and written \(r^{-1}(x) = y\)

Example 1 The rule \(r_1\) with Domain set \(D = \{ -2, -1, 0, 1, 2 \}\) and the Range set \(R = \{ -1, 1\},\) define \(r_{1}(x)\) with the statement “if \(x\) is in \(Z\) then \(r_{1} (x) = 1\) for \(x > 0\) and \(r_{1}(x) = -1\) for \(x \leq 0\) is not an injection.

Example 2 Define the rule \(r_{2}\) from the discrete set of all positive integers \(Z^+ = \{ 1, 2, 3, \ldots \}\) to the discrete set of all rational numbers \(\mathbb{Q} = \{q \vert q = \frac{x}{y} \text{ with } x,y \in \mathbb{Z} \text{ and } y \neq 0\}\), with the following: \(r_{2}(x) = \frac{1}{x}.\) This rule is an injection.
