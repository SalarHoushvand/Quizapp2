Function properties (homomorphism)

For a rule \(r\) with Domain set \(D\) and range set \(R\) where both \(D\) and \(R\) have operations \(op_D\) and \(op_R\) defined on them, the rule is said to be a homomorphism if the rule is such that for any \(x\) and \(y\) in \(D,\) r maps \(x\) \(op_D\) \(y\) to the element \(r(x)\) \(op_R\) \(r(y)\) in \(R.\)

Example 1 For the rule \(r\) with Domain \(\mathbb{Z} = \{ \ldots, -3, -2, -1, 0, 1, 2, 3, \ldots \}\) and Range set \(\mathbb{Z} = \{ \ldots, -3, -2, -1, 0, 1, 2, 3, \ldots \}\) where the operations defined on the integers, \(\mathbb{Z}\) are standard integer multiplication and addition.

Define \(r(x)\) with \(r(x) = x^2.\) Then it can be shown that \(r(x + y) = (x + y)^2 \neq r(x) + r(y)= x^2 + y^2\) for all choices of integers \(x\) and \(y.\) In particular let \(x = 2\) and \(y = 3,\) then \(r(2 + 3) = 25\) but \(r(2) + r(3) = 4 + 9 = 13.\) Thus, \(r\) is not a homomorphism.

Example 2 For the rule \(r\) with Domain \(\mathbb{Z} = \{ \ldots, -3, -2, -1, 0, 1, 2, 3, \ldots \}\) and Range set \(\mathbb{Z}_2 = \{[0], [1]\}\) where the equivalence class \([0]\) is the set of all integers that are even and the equivalence class \([1]\) is the set of all odd integers. The operations defined on the integers, \(\mathbb{Z}\) are the standard integer multiplication and addition. For the Range set \(\mathbb{Z}_2,\) the integers mod 2, the operatons use the fact that eveneven = even, evenodd = even, odd*odd = odd, even+odd = odd, even + odd = odd and odd + odd = even:

+ [0] [1]
[0] [0] [1]
[1] [1] [0]
* [0] [1]
[0] [0] [0]
[1] [0] [1]

Define \(r(x)\) with \(r(x) = x^2.\) Then it can be shown that \(r(x + y) = r(x) + r(y)\) so that \(r\) is a homomorphism.
