Function composition

For two functions, \((D_f, R_f, f)\) and \((D_g, R_g, g),\) or simply \(f\) and \(g,\) we can define a new function as the composition of \(f\) and \(g,\) denoted \((D_g, R_f, f \circ g ),\) if \(R_g\) is a collection of elements that are also in the collection of elements \(D_f.\) The rule \(f \circ g\) assigns to each \(x\) in \(D_g\) the element \(f(g(x))\) in \(R_f.\)

Example 1: Take the two functions \((\mathbb{Z}, \mathbb{Z}, f)\) and \((\mathbb{Z}, \mathbb{Z}, g)\) with the rules \(f(x) = x^2\) and \(g(x) = 2.\) The Range set of \(g\) is equal to the Domain set of \(f,\) and so the composition of these two functions exists and is \((\mathbb{Z}, \mathbb{Z}, f \circ g)\) where \(f\circ g (x) = 4.\)

The function \(f\) in the previous example is considered the “outer” function or a wrapper. The “inner” function is \(g.\) The inner function begins with an element \(x\) in \(D_g\) and assigns it to an element \(g(x)\) in \(R_g \subseteq D_f.\) Then the rule \(f\) maps the element \(g(x)\) to an element \(f(g(x))\) in the set \(R_f.\) The Image set, \(f \circ g,\) will not necessarily be equal to \(Im_f.\) However \(Im_{f \circ g} \subseteq Im_{f}.\)

Example 2
Let \(\mathbb{Z}_{2x2}-\{\begin{bmatrix} a & b \\c & d\end{bmatrix} \vert ad-bc \neq 0\}\) be the set of all matrices with 2 rows and 2 columns and integer entries where the difference of the product of the terms in the main diagonal and the anti-diagonal is 0. Then the function \(n,\) which normalizes the entries of a matrix \(M \in D_n\), is defined as follows:

\[n(M) = n(\begin{bmatrix} a & b \\ c & d\end{bmatrix}) = \frac{1}{ad - bc} \begin{bmatrix} a & b \\ c & d \end{bmatrix} = \begin{bmatrix} \frac{a}{ad-bc} & \frac{b}{ad-bc} \\ \frac{c}{ad-bc} & \frac{d}{ad-bc} \end{bmatrix}.\]

This function \(n\) will serve as the inner function in our composition since we desire to apply this function to the set of all 2-by-2 matrices. The outer function will be \(d\) and have its Domain set \(D_d = \mathbb{Z}_{2x2}\) and Range set \(R_d = \mathbb{Z}.\) All that is required for the function \(d\circ n\) to exist is \(R_n \subseteq D_d.\) The rule \(d\) maps a matrix \(N \in \mathbb{Z}_{2x2}\) to the determinant \(d(N)\):

\(d(N) = d(\begin{bmatrix} a & b \\ c & d\end{bmatrix}) = ad - bc.\) Then \(d \circ n (M) = 1\) for all \(M \in D_n\) so that the Range set for \(d \circ n\) is \(R_{d \circ n} = \{1\}.\)
